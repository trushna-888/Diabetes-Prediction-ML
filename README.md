# 🩺 Diabetes Prediction using Machine Learning

📌 Project Overview

This project predicts whether a person has diabetes based on medical parameters using Machine Learning.

---

 📂 Dataset

- Pima Indians Diabetes Dataset
- File: `diabetes.csv`

---

 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

---

📊 Features Used

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age

---

 🎯 Goal

To predict whether a patient is diabetic or not.

 ▶️ How to Run

```bash
python diabetes_prediction.py
```
